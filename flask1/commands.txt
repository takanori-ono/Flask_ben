[仮想を作成] gitには仮想環境をコミットしないほうがいいとのこと。そのため、clone後は作成する。
py -m venv .venv

[仮想に入る]
.venv/scripts/activate.bat

[仮想から出る]
.venv/scripts/deactivate.bat

[外部パッケージインストール]
pip install flask

[外部パッケージの一覧]
pip freeze > requirements.txt

[パッケージのインストール（requirements.txt）]
pip install -r requirements.txt

[オフライン環境用にパッケージファイルのダウンロード]
pip download -r requirements.txt -d ./whl_files

[オフライン環境でのパッケージインストール]
pip install --no-index --find-links=./whl_files -r requirements.txt
